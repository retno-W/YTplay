YTPlay

YTPlay adalah aplikasi berbasis Flask untuk membuat halaman pemutar video dari berbagai sumber video online maupun file lokal. Setiap video yang ditambahkan akan mendapatkan URL pendek yang dapat dibagikan.

Fitur
Embed video dari berbagai platform:
YouTube
Vimeo
Dailymotion
Twitch
TikTok
Facebook
Google Drive
Streamable
Bilibili
OK.ru
Upload video dan audio lokal
URL share unik untuk setiap video
Statistik jumlah pemutaran
Riwayat video
Video terkait (related videos)
Dashboard admin dengan HTTP Basic Authentication
Manajemen menu navigasi
Edit judul video
Hapus video dan data
Database SQLite
Upload file hingga 1 GB
Konversi otomatis HEVC/H.265 ke H.264 menggunakan FFmpeg
Proses konversi berjalan di background
Persyaratan
Python 3.8+
Flask
Flask-HTTPAuth
yt-dlp
FFmpeg
SQLite
